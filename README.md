**O que faz:** Lê PDFs, entende o conteúdo e responde suas perguntas
**Tecnologias:** Python, LangChain, ChromaDB, OpenAI
**Status:** Funcionando (e me orgulho disso!)

**Tecnologia:** Python puro (sem frescura)
**Status:** Uso todo dia
